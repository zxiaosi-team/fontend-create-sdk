# main-react

# 文件来自哪里？（以 `React` 举例）

## 本地安装

- 安装依赖

```bash
npm install react@18
```

- 文件位置

```bash
node_modules/react/umd/index.js
```

## CDN

- [react](https://unpkg.com/react@18/umd/react.development.js)
- [react-dom](https://unpkg.com/react-dom@18/umd/react-dom.development.js)

# 包与文件对应关系

| 包名           | NPM                  | CDN                                                           |
| -------------- | -------------------- | ------------------------------------------------------------- |
| `react@18`     | `npm i react@18`     | [react@18](https://app.unpkg.com/react@18/files/umd)          |
| `react@19`     | `npm i umd-react@19` | [umd-react@19](https://app.unpkg.com/umd-react@19/files/dist) |
| `react-dom@18` | `npm i react-dom@18` | [react-dom@18](https://app.unpkg.com/react-dom@18/files/umd)  |
| `react-dom@19` | `npm i umd-react@19` | [umd-react@19](https://app.unpkg.com/umd-react@19/files/dist) |
